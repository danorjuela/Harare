# Harare 

## Prerequisites

- [Docker](www.docker.com)
- [Node.js](nodejs.org) >= 10.13.0

## Installation

```bash
$ npm install
```

## Prepare environment

```bash
$ npm run start-env:ps
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
